ROOK-CLF-9M Interactive Chess Analysis

Explore how a 9M parameter language model achieves strategic chess reasoning without search. This interactive demo allows you to analyze any chess position and see the model's move predictions and confidence scores.

Key Achievements:

  • 49% best move accuracy (Google DeepMind benchmark)
  • 57% checkmate-in-one accuracy
  • Searchless strategic reasoning
  • Real-time browser inference

RookWorld Interactive Space

Experience the unified agent+environment model that combines chess policy and world simulation in a single transformer. Play against the model or watch it play against itself.

Features:

  • Self-play without external engines
  • 99.9% environment simulation accuracy
  • Chain-of-thought reasoning traces
  • Multi-task learning demonstration
